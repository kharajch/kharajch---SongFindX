# 🎵 SongFindX

**Find and Download Songs** — A modern music discovery web app built with Next.js.

🔗 **Live Demo:** [kharajch-songfindx.vercel.app](https://kharajch-songfindx.vercel.app)
📂 **Repository:** [github.com/kharajch/kharajch---SongFindX](https://github.com/kharajch/kharajch---SongFindX)

---

## ✨ Features

- 🔍 **Search Songs** — Instantly search and find songs by name.
- 🌐 **Filter by Language** — Browse songs in Bengali, Hindi, English, and more.
- 🎧 **Filter by Genre** — Explore Romantic, Energetic, Sad, and other genres.
- 🎤 **Filter by Artist** — Discover tracks from your favorite singers.
- 🧭 **Explore Page** — Dive deeper into languages, genres, and artists.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/kharajch/kharajch---SongFindX.git

# Navigate into the project
cd kharajch---SongFindX

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
